# Um Cabeçalho Básico

Use regras do flexbox para criar este estilo muito comum de cabeçalho de página web. O benefício de usar flex aqui é que tudo deve ser _flexível_. Veja as duas capturas de tela abaixo para ter uma ideia de como deve escalar com sua tela. Além das regras flex, você também vai querer adicionar algumas regras para margem e preenchimento. (Dica: `ul`s têm alguma margem/padding padrão que você precisará lidar.)

### Verificação Própria
- Há espaço entre todos os itens e a borda do cabeçalho (a quantidade específica de px não importa aqui).
- O logo está centrado verticalmente e horizontalmente.
- Os itens da lista estão horizontais e centrados verticalmente dentro do cabeçalho.
- Os left-links e right-links são empurrados totalmente para a esquerda e direita, e ficam na borda do cabeçalho quando a página é redimensionada.
- Sua solução não usa floats, inline-block ou posicionamento absoluto.

- Nota: Para este exercício, é completamente aceitável não corresponder à família de fontes.
