# Outro estilo comum de cabeçalho

Estamos começando a introduzir um pouco mais de CSS que você ainda não viu. Não se preocupe com isso por enquanto; queremos apenas que as coisas fiquem um pouco mais bonitas, e esse CSS não vai interferir na sua tarefa.

Para este exercício, você provavelmente precisará editar um pouco o HTML. Frequentemente com flexbox é preciso adicionar contêineres ao redor dos elementos para fazê-los ir para onde você precisa. Neste caso, você provavelmente vai querer separar os itens que ficam à esquerda e à direita do cabeçalho.

Assim como você fez no exercício `02-flex-header`, você vai aninhar contêineres flex uns dentro dos outros.

## Resultado desejado
Assim como no último exemplo, este precisa ser flexível no centro, com itens empurrados para a esquerda e para a direita.
