# Uma página inteira!

O Flexbox é útil para dispor páginas inteiras, bem como os componentes menores com os quais já trabalhamos. Para este exercício, estamos deixando um pouco mais de trabalho para você, com algumas coisas que você pode não ter encontrado ainda. É perfeitamente aceitável pesquisar no Google coisas das quais você não tem certeza!

### Dicas
- Você pode querer pesquisar algo como `CSS remove list bullets`. Fizemos isso para você em exemplos anteriores, mas não aqui. Viva o aprendizado.
- Descobrir como estilizar links em CSS pode ajudar a se livrar daquela decoração chata de sublinhado...
- Adicionamos `height: 100vh` ao `body`... isso faz com que o body tenha exatamente a mesma altura da viewport. Para fixar o footer na parte inferior, você precisará usar flex e alterar a direção para coluna.

### Verificação Própria

- O cabeçalho está no topo da página, o footer na parte inferior, e eles permanecem no lugar se você redimensionar a tela.
- O cabeçalho e o footer têm padding.
- Os links no cabeçalho e footer são empurrados para os lados.
- Há espaço entre os links no cabeçalho e footer.
- O footer tem um fundo cinza claro (`#eeeeee`).
- O logo, input e botões estão centrados na tela.
- Os botões têm uma quantidade apropriada de padding.
- Há espaço entre o logo, input e botões.
